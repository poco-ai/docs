# 一键初始化脚本

如果你是第一次在本地使用内置 RustFS 存储启动，推荐使用初始化脚本。

## 前置条件

- Docker Desktop 或 Docker Engine
- Docker Compose v2
- 至少 2 核 CPU / 4 GB 内存
- 若镜像为私有，需要先执行 `docker login ghcr.io`

## 命令

```bash
./scripts/quickstart.sh
```

## 它会完成的事情

- 在需要时把 `.env.example` 复制为 `.env`
- 写入 Docker 与模型相关环境变量
- 创建本地数据目录
- 默认拉取镜像并启动服务
- 通过 `rustfs-init` 创建配置好的 S3 bucket
