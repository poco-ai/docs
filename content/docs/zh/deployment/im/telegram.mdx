# Telegram 集成

## 创建机器人

1. 打开 `@BotFather`
2. 执行 `/newbot`
3. 保存 `TELEGRAM_BOT_TOKEN`

## 建议变量

```bash
TELEGRAM_BOT_TOKEN=<你的-bot-token>
TELEGRAM_WEBHOOK_SECRET_TOKEN=<随机长字符串>
```

## 启动 IM 服务

```bash
docker compose --profile im up -d im
```

## 注册 webhook

把 Telegram webhook 指向公网 HTTPS 域名下的 `/api/v1/webhooks/telegram`，如果启用了密钥校验，`secret_token` 需要与环境变量保持一致。
