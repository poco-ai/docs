# 常用操作

## 默认地址

- Frontend：`http://localhost:3000`
- Backend：`http://localhost:8000`
- Executor Manager：`http://localhost:8001`
- IM：启用后为 `http://localhost:8002`

## 日常操作

- 使用 `docker compose logs -f <service>` 查看日志
- 使用 `docker compose restart <service>` 重启服务
- 通过配置页面定位正确的 `.env` 设置项
