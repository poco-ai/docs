# Common Operations

## Default URLs

- Frontend: `http://localhost:3000`
- Backend: `http://localhost:8000`
- Executor Manager: `http://localhost:8001`
- IM: `http://localhost:8002` when enabled

## Routine tasks

- Use `docker compose logs -f <service>` to inspect logs
- Restart services with `docker compose restart <service>`
- Use the configuration pages to locate the right `.env` settings
