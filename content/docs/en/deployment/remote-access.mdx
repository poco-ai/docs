# Remote Access with Tailscale

Tailscale can be used to expose your local Poco frontend for remote use.

## Typical flow

1. Log in to Tailscale and join your tailnet
2. Expose port `3000` with Tailscale Serve
3. Access the frontend from approved remote devices
